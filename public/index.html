🧱 LEGO Diary Reader

📸 Image Gateway

🌐 GitHub Pages Image Gateway

This directory serves as a decentralized image gateway for LEGO diary entries. Images are stored with their IPFS hash as the filename, allowing seamless fallback between IPFS and GitHub Pages.

📁 Directory Structure:

/lego-diary-reader/public/
├── images/
│ ├── QmXXXXXX.png (IPFS hash filenames)
│ ├── bafkreiXXXX.png
│ └── ...
└── index.html (this file)

🔗 How It Works

Image Access Priority:

  1. Local Development: http://localhost:3000/images/HASH.png
  2. GitHub Pages: https://username.github.io/repo/lego-diary-reader/public/images/HASH.png
  3. IPFS Gateway: https://gateway.pinata.cloud/ipfs/HASH
